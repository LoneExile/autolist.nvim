# autolist.nvim
Minimal automatic list continuing for ordered and unordered lists for neovim in lua

supports:
* ordered (n. '...')
* unordered
```
	- content
	* content
	+ content
```
* autodeletes empty list bullets
* indented lists with `<c-t>` and `<c-d>`

doesn't support:
* renumbering of lists
